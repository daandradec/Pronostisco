Paso 1

Una MRP (sistema de planificación de recursos), es una herramienta para la gestión de sus recursos para sus procesos de producción, así que el paso 1 para usar nuestro sistema es abstraer su Producto principal y subdividirlo en componentes y materias primas, que junto a datos que pueda ingresar usted obtendra unos resultados para tomar decisiones. Por ejemplo si quiere saber cuanto le cuesta hacer una camisa, crea el producto camisa y sus partes, e ingresa datos como costos, y obtiene los resultados.

Paso 2

Usted dispone de un grafico de arbol para ver como se compone su producto. Tambien dispone de un campo de texto para modificar el nombre de su producto como desee, y mas abajo los bloques de creacion de componentes y materias primas.

Paso 3

Utilice los botones para crear componentes y materias primas que conforman su producto, haga click en el nombre para modificar el nombre de ese componente o materia, y tambien utiliza el campo numerico para definir la cantidad de ese componente o materia prima que tiene con respecto al padre al que esta asociado.
